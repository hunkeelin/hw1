## Introduction
Run the code with the sample .txt files in the same directory

### Usage
I've include the binary for each platform. Windows,Darwin and Linux. 
E.g
```
./linux.ipynb
  -dataset string
        dataset (default "data.txt")
  -negative string
        the list of negative words (default "neg.txt")
  -positive string
        the list of positive words (default "pos.txt")
  -stop string
        the list of stop words (default "stop.txt")
```
